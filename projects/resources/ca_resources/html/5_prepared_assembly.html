Готовые сборки

Назначение

  • Для копирования и переименование специально подготовленных сборок .iam

  • Добавление в список программы новых подготовленных сборок

Пошаговое руководство

1. Выберите нужную сборку из списка

2. В поле Заменить на введите строку, которая заменит подстроку Искать в строке Текущее имя сборки и отобразить в Имя сборки

3. Далее нажмите Продолжить

4. После окончания процесса выберите папку назначения для готовый сборки
